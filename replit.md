# Система учета картриджей

## Обзор
Веб-приложение для учета служебных записок по картриджам для принтеров. Построено на Python + FastAPI с PostgreSQL базой данных и Bootstrap интерфейсом.

## Структура проекта
```
/
├── main.py              # Основное FastAPI приложение
├── src/
│   ├── database.py      # Подключение к PostgreSQL
│   ├── models.py        # SQLAlchemy модели
│   └── schemas.py       # Pydantic схемы
├── templates/           # HTML шаблоны (Jinja2)
│   ├── base.html
│   ├── dashboard.html
│   ├── cartridges.html
│   ├── warehouses.html
│   ├── departments.html
│   ├── employees.html
│   ├── service_notes.html
│   └── reports.html
└── seed_data.py         # Демо-данные
```

## Функционал
1. **Картриджи** - управление картриджами (артикул, модель, тип, цвет, статус)
2. **Склад** - склады и ящики с отслеживанием остатков
3. **Отделы** - структура компании
4. **Сотрудники** - база сотрудников
5. **Служебки** - создание и учет служебных записок на выдачу картриджей
6. **Отчеты** - остатки, статистика по отделам, история движения

## Запуск
```bash
python main.py
```
Сервер запускается на порту 5000.

## База данных
Используется PostgreSQL через SQLAlchemy ORM. Таблицы создаются автоматически при запуске.

## Бизнес-логика
- Автоматическая нумерация служебок: КАРТ-YYYY-NNN
- При выдаче уменьшается остаток в ящике
- При возврате увеличивается остаток
- История всех перемещений картриджей
